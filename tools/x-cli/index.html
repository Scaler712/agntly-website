x-cli

A reverse-engineered CLI for X (Twitter). Fetch timelines, search tweets, pull user profiles and follower lists — all through X's internal GraphQL API. No API key required. No paid tier. Completely free.

View on GitHub How It Works
The Problem

X's API is paywalled.
Agents are locked out

X's official API costs $100/month for basic access and $5,000+/month for anything meaningful. Most endpoints are restricted, rate limits are aggressive, and the free tier is practically useless. For AI agents that need to read timelines, search content, or analyze profiles — the door is closed.

# The official X API experience
1. Apply for API access (wait days)
2. Pay $100/mo for Basic tier
3. Get 100 tweets/month read limit
4. No search. No timelines. No followers.
5. Upgrade to $5,000/mo for "Pro"
 
# Or just don't use X data at all
# Most teams choose this option
# With x-cli: full access, zero cost
$ x-cli auth login
Opening Chrome for X login...
✓ Authenticated as @agntly
 
$ x-cli search "AI agents" --type latest
Fetching latest results...
✓ 20 tweets returned
 
$ x-cli timeline user @elonmusk --count 50
✓ 50 tweets returned
The Solution

We reverse-engineered
X's internal API

x-cli uses the same internal GraphQL API that the X web app itself uses. We reverse-engineered its endpoints, authentication flow, and query structure — giving agents full read access to timelines, search, profiles, and social graphs without paying for a single API key.

Capabilities

Everything agents need
from X

Timelines

Fetch the home timeline or any user's tweet history. Supports pagination with cursors and auto-paginate mode for bulk data collection across multiple pages.

x-cli timeline user @elonmusk --all --max-pages 5

Search

Full search across X with multiple result types: top, latest, people, and media. The same search power the web app provides — available from the command line.

x-cli search "breaking news" --type latest

User Profiles

Pull complete user profiles with bio, follower counts, verification status, and all public metadata. Structured data ready for agent consumption.

x-cli user get @agntly

Social Graph

Retrieve follower and following lists for any public account. Paginated and structured — ideal for agents building relationship maps or audience analysis.

x-cli followers @agntly --count 50

Tweet Lookup

Fetch any individual tweet by ID or full URL. Get complete tweet data including engagement metrics, replies context, and media attachments.

x-cli tweet get https://x.com/user/status/123...

JSON Output

Every command supports --json for raw API output and --verbose for debugging. Designed for agent pipelines — structured data in, structured data out.

x-cli timeline home --json | jq '.tweets'
Architecture

Reverse-engineered with precision

x-cli intercepts X's internal GraphQL endpoints — the same ones the web app uses in production. Browser-based auth captures session tokens, and the CLI replays them against X's undocumented API surface.

# Architecture
 
cmd/ # Cobra CLI commands
auth.go # Browser login flow (Chrome)
timeline.go # Home & user timelines
search.go # Search across all content types
user.go # Profile lookup
tweet.go # Individual tweet fetch
followers.go # Follower & following lists
 
internal/
api/ # GraphQL client & endpoint registry
auth/ # Browser auth & credential storage
models/ # Domain types (Tweet, User, Timeline)
output/ # Pretty-print & JSON formatting
